# Task 1 - Get Local Maxes

```
Да се напише функция с параметър масив от числа, 
която намира локалните максимуми в масива и разпечатва масива, 
като максимумите са заградени в скоби. 

Локален максимум е елемент, който е по-голям от съседните си елементи.

Пример:
1, 2, 3, 2, 0, 6, 5 -> 1 2 [3] 2 0 [6] 5
```
