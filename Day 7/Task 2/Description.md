# Task 2 - RGB To Hex Representation

```
Да се напише функция rgb, която получава 3 стойности в интервал 0-255 включително.
И връща стринг, в който стойностите са в 16-чен код и са залепени по редна на параметрите.

Пример:
(255, 255, 255) -> FFFFFF
(255, 255, 254) -> FFFFFЕ
(0, 0, 0) -> 000000
(148, 0, 211) -> 9400D3
```
