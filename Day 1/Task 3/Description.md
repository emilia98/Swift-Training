# Task 3 - Format String

```
Приемаме, че при печат на стринг символа # изтрива предходния. 
Това означава, че стринг а"a#bc#d" ще бъде отпечатана така "bd".
Напишете програма, която печата така.
Примери:
"abd##c"      ==>  "ac"
"abc##d######"  ==>  ""
"#######"       ==>  ""
""              ==>  ""
```
