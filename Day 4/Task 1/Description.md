# Task 1 - Comma Separated Values (CSV)

```
Парсиране на comma separated values (CSV):
Да се  напише функция, която получава като аргумент стринг със стойности,
разделени със запетая и връща масив от съответните стойности.

Забележки:
1. Преди и след дадена стойност може да има интервали;
2. Някои стойности могат да бъдат в единични или двойни кавички;
3. Стойностите в кавички съдържа символ за кавички, 
   ако преди него има обратна наклонена черта \ 
   той се приема за част от стринга, а не за негов край

Примери:
1, "аааа", 5 -> ["1", "aaaa", "5"]
12, 'aaaa', 3 -> ["12", "aaaa", "3"]
"aa\"bb", test -> ["aa\"bb", "test"]
```
