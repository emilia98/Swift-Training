# Task 2 - Point Colors In Rectangles

```
Даден е масив с правоъгълници, за всеки от който са зададени координатите на горния ляв и долния десен ъгъл, 
както и цвета му. 
Правоъгълницте се рисуват върху екрана един след друг, като могат да се препокриват. 
Първо се рисува последния елемент в масив, върху него е следващият и най-отгоре е първият елемент.

Да се напише функция, която получава координати на точка и връща цвета на тази точка, 
ако тя е попада върху правоъгълник, ако не undefinеd. 
Цветовете са зададени като стрингове.
```
